Monos 테마 한글 업로드 테스트입니다. `백틱 표시`는 다음과 같이 출력됩니다.
코드 출력 예시입니다.

{% highlight ruby %}
def print_hi(name)
  puts "Hi, #{name}"
end
print_hi('Tom')
#=> prints 'Hi, Tom' to STDOUT.
{% endhighlight %}
